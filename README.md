<h1 align="center">
pubm
</h1>

<p align="center">
publish manager for multiple registry (JSR, npm and private registry)
<p>

## Why
It is designed to be easy to managing publish to multiple registry.

## Usage
```bash
pubm publish
```

## Config file for publish
You can have either package.json or jsr.json.
