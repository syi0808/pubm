---
title: Reference API de configuration
description: Champs supportes de `pubm.config.*`, valeurs par defaut et notes d'usage.
---

Cette reference decrit la forme typee de `pubm.config.*` exposee par `defineConfig()`.

Champs les plus importants aujourd'hui:

- `packages`
- `plugins`
- `versioning`, `fixed`, `linked`
- `snapshot.useCalculatedVersion`

Les registries sont inferees depuis les fichiers de manifeste. `packages[].registries` permet de les surcharger ou d'ajouter des registries privees.

## Defauts principaux

- `versioning: "independent"`
- `branch: "main"`
- `tag: "latest"`
- `contents: "."`
- `saveToken: true`
- `releaseDraft: true`

## Champs utiles

- `packages`
- `fixed`
- `linked`
- `plugins`
- `validate`
- `snapshot`
