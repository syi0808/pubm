title: Reference API des plugins
description: Ajouter hooks, commandes, registres et ecosystemes a pubm.
---

Le systeme de plugins permet d'etendre `pubm` sans forker le pipeline central.

## `PubmPlugin`

```ts
export interface PubmPlugin {
  name: string;
  registries?: Registry[];
  ecosystems?: Ecosystem[];
  hooks?: PluginHooks;
  commands?: PluginCommand[];
}
```

## `Ctx`

```ts
export interface Ctx extends ResolvedOptions {
  promptEnabled: boolean;
  cleanWorkingTree: boolean;
  pluginRunner: PluginRunner;
  releaseContext?: ReleaseContext;
}
```

Champs utiles :

- `version`
- `versions`
- `packages`
- `tag`
- `preview`
- `preflight`
- `ci`
- `changesetConsumed`
