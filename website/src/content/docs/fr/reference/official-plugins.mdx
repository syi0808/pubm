---
title: Plugins officiels
description: Plugins supportes et maintenus par le projet pubm.
---

Les plugins officiels sont maintenus avec `pubm` et conviennent quand vous voulez une integration standardisee.

## `@pubm/plugin-brew`

Synchronise une formule Homebrew avec chaque release.

## `@pubm/plugin-external-version-sync`

Garde les fichiers hors manifest alignes avec la version du package.

Pour les details d'extension, lire [Plugins API Reference](/pubm/fr/reference/plugins/).
