title: Integration avec les agents de code
description: Integrer le plugin pubm dans Claude Code, Codex et d'autres agents de code.
---

Ce guide s'adresse aux equipes qui utilisent deja des agents de code et veulent brancher les workflows de release `pubm` dans cet environnement existant.

## Commencer par le setup

Si le depot n'est pas encore cable, l'action la plus importante n'est pas publish mais setup.

Utilisez d'abord `publish-setup`. Ce skill gere :

- l'installation du bundle `pubm plugin`
- la generation ou mise a jour de la config `pubm`
- le branchement de la CI
- l'activation des changesets

Donnez a l'agent le lien [INSTALLATION.md](/pubm/plugins/pubm-plugin/INSTALLATION.md) pour qu'il recupere le bundle et suive le setup versionne dans le depot.

## Installation Claude Code

```text
/plugin marketplace add your-org/claude-plugins
/plugin install plugin-name@marketplace-name
```

Si vous n'utilisez pas de marketplace, fournissez a l'agent le lien `INSTALLATION.md`.
