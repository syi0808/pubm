`pubm` ist CLI-first, aber echte Projekte brauchen schnell repository-weite Konfiguration uber `pubm.config.*`.

## Unterstutzte Konfigurationsdateien

1. `pubm.config.ts`
2. `pubm.config.mts`
3. `pubm.config.cts`
4. `pubm.config.js`
5. `pubm.config.mjs`
6. `pubm.config.cjs`

## Registry-Ableitung

| Manifest | Abgeleitete Registry |
|---|---|
| `package.json` | `npm` |
| `jsr.json` | `jsr` |
| `Cargo.toml` | `crates` |

## Wichtige Felder

- `packages`
- `plugins`
- `versioning`
- `fixed`
- `linked`
- `ignore`
- `snapshotTemplate`
