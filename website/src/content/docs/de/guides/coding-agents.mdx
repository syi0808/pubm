Dieses Dokument richtet sich an Teams, die bereits Coding Agents einsetzen und `pubm` in diese bestehende Umgebung integrieren wollen.

## Mit Setup beginnen

Wenn das Repository noch nicht verkabelt ist, ist die wichtigste Aktion nicht Publish, sondern Setup.

Nutze zuerst `publish-setup`. Gib dem Agent den Link [INSTALLATION.md](/pubm/plugins/pubm-plugin/INSTALLATION.md), damit er das Bundle holen und den eingecheckten Setup-Flow befolgen kann.

## Claude Code Installation

```text
/plugin marketplace add your-org/claude-plugins
/plugin install plugin-name@marketplace-name
```
