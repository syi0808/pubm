---
title: Changesets
description: Release-Absicht festhalten, Changelogs erzeugen und Versions-Bumps steuern.
---

`pubm` nutzt Dateien in `.pubm/changesets/` als dauerhafte Quelle dafur, was als Nachstes releast werden soll.

## Grundkommandos

```bash
pubm init
pubm init --changesets
pubm changesets add
pubm changesets status --verbose
pubm changesets version --dry-run
```

## Verbrauch im Release-Flow

Wenn du `pubm` startest, werden ausstehende Changesets im Version-Schritt konsumiert. `pubm changesets version` ist fur kontrollierte Sonderfalle weiter verfugbar, aber nicht zwingend vor `pubm` oder `pubm --ci`.
