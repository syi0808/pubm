---
title: CLI Reference
description: Befehle, Flags und Ausfuhrungsmodi der pubm CLI.
---

Wichtigster Befehl:

```bash
pubm
```

Beispiele:

```bash
pubm patch
pubm minor --preview
pubm 1.8.0
pubm --preflight
pubm --ci
```

Wichtige Flags:

- `--preview`
- `--branch <name>`
- `--tag <name>`
- `--contents <path>`
- `--registry <registries>`
- `--test-script <script>`
- `--build-script <script>`
- `--snapshot [tag]` — Veröffentlicht eine temporäre Snapshot-Version (Standard-Tag: snapshot)
