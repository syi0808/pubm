---
title: Config API Reference
description: Unterstutzte Felder in `pubm.config.*`, Standardwerte und Hinweise.
---

Wichtige Felder:

- `packages`
- `plugins`
- `versioning`
- `fixed`
- `linked`
- `snapshot.useCalculatedVersion`

Registries werden aus den Manifest-Dateien abgeleitet. `packages[].registries` erlaubt Uberschreibung oder private Registries.

Wichtige Defaults:

- `versioning: "independent"`
- `branch: "main"`
- `tag: "latest"`
- `contents: "."`
