The `pubm` marketplace is a centralized registry for discovering and sharing plugins, specifically designed for agentic workflows like **Claude Code**.

## Marketplace Structure

The marketplace is defined by a `marketplace.json` file. This file lists all available plugins and their metadata.

```json
{
  "name": "pubm-marketplace",
  "owner": {
    "name": "pubm"
  },
  "plugins": [
    {
      "name": "pubm-plugin",
      "source": "./plugins/pubm-plugin",
      "description": "Publish packages to multiple registries...",
      "version": "0.3.0",
      "author": {
        "name": "pubm"
      },
      "keywords": ["publish", "npm", "jsr", "release"],
      "category": "devtools"
    }
  ]
}
```

## Plugin Entry Fields

| Field | Description |
|-------|-------------|
| `name` | Unique identifier for the plugin. |
| `source` | Path to the plugin's source code or skill definitions. |
| `description` | A concise summary of what the plugin does. |
| `version` | Current version of the plugin. |
| `author` | Creator of the plugin. |
| `keywords` | Tags for discoverability. |
| `category` | Classification (e.g., `devtools`, `productivity`). |

## Contributing a Plugin

To list your plugin in the marketplace:

1.  **Develop your plugin**: Follow the [Plugins API Reference](/pubm/reference/plugins) to create a standard `pubm` plugin or define agentic skills.
2.  **Add to `marketplace.json`**: Create a PR to the `pubm` repository adding your plugin metadata to the `plugins` array.
3.  **Validation**: The `pubm` team will review your plugin for safety and quality before merging.

## Agentic Skills

Plugins in the marketplace often include "Skills" — specialized instructions that AI agents (like Claude) use to perform tasks.

- **Portable**: Skills are written in Markdown and can be used across different agent platforms.
- **Tool-Integrated**: They define which CLI tools the agent can use to fulfill the skill's objective.
- **Safety-First**: Skills often enforce a "preview-first" workflow to ensure the user is always in control.
