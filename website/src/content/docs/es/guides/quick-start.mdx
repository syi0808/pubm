---
title: Inicio rápido
description: Configura pubm en minutos y publica tu primer release.
---

`pubm` está pensado para empezar rápido. Si ya tienes un monorepo o un paquete con manifest, normalmente basta con instalarlo y ejecutar el flujo interactivo.

## Instalar

```bash
npm install -D pubm
pnpm add -D pubm
bun add -d pubm
```

## Inicializar

```bash
pubm init
```

Esto crea la configuración mínima y los directorios necesarios para empezar.

## Publicar

```bash
pubm
```

Si hay changesets pendientes, `pubm` los consume durante la fase de versionado del pipeline de release. Para configuración, lee [Configuration](/pubm/es/guides/configuration/). Para CI, tokens y comportamiento de preparación de CI, lee [CI/CD](/pubm/es/guides/ci-cd/). Para monorepos, lee [Monorepo](/pubm/es/guides/monorepo/). Para flags y subcomandos exactos, lee la [CLI Reference](/pubm/es/reference/cli/).
