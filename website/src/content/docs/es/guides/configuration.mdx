`pubm` es CLI-first, pero los proyectos reales necesitan rapido una configuracion a nivel de repositorio con `pubm.config.*`.

## Config minima

`pubm` infiere los registros desde los archivos de manifiesto. La config puede quedar vacia en la mayoria de proyectos:

```ts
import { defineConfig } from "@pubm/core";

export default defineConfig({});
```

Reglas de inferencia:

| Manifiesto | Registro inferido |
|---|---|
| `package.json` | `npm` |
| `jsr.json` | `jsr` |
| `Cargo.toml` | `crates` |

Campos comunes:

- `packages`
- `plugins`
- `versioning`
- `fixed`
- `linked`
