---

`pubm` es CLI-first, pero los proyectos reales necesitan rapido una configuracion a nivel de repositorio con `pubm.config.*`.

## Archivos soportados

1. `pubm.config.ts`
2. `pubm.config.mts`
3. `pubm.config.cts`
4. `pubm.config.js`
5. `pubm.config.mjs`
6. `pubm.config.cjs`

## Reglas de inferencia

| Manifiesto | Registro inferido |
|---|---|
| `package.json` | `npm` |
| `jsr.json` | `jsr` |
| `Cargo.toml` | `crates` |

## Campos comunes

- `packages`
- `plugins`
- `versioning`
- `fixed`
- `linked`
- `ignore`
- `snapshotTemplate`
