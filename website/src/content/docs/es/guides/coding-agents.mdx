Este documento esta dirigido a equipos que ya usan agentes de codigo y quieren integrar los workflows de release de `pubm`.

## Empezar por setup

Si el repositorio todavia no esta conectado, la accion mas importante no es publicar sino configurar.

Usa primero `publish-setup`. Dale al agente el enlace [INSTALLATION.md](/pubm/plugins/pubm-plugin/INSTALLATION.md) para que obtenga el bundle y siga el flujo de setup versionado.

## Instalacion de Claude Code

```text
/plugin marketplace add your-org/claude-plugins
/plugin install plugin-name@marketplace-name
```
