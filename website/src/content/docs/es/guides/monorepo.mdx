---
title: Monorepo
description: Publicar paquetes del workspace en el orden correcto y con la estrategia de version adecuada.
---

`pubm` trata los monorepos como un caso principal y puede detectar workspaces de JavaScript, Rust y Deno.

## Flujo tipico

```bash
pubm init --changesets
pubm changesets add
pubm
```
