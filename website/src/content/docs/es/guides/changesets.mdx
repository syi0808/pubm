---
title: Changesets
description: Registrar la intencion de release, generar changelogs y controlar los bumps de version.
---

`pubm` usa `.pubm/changesets/` como el registro durable de lo que debe liberarse despues.

## Comandos basicos

```bash
pubm init
pubm init --changesets
pubm changesets add
pubm changesets status --verbose
pubm changesets version --dry-run
```

## Consumo en el flujo de release

Cuando ejecutas `pubm`, los changesets pendientes se consumen durante la etapa de version. `pubm changesets version` sigue disponible para flujos controlados, pero no es obligatorio antes de `pubm` o `pubm --ci`.
