---
title: CLI Reference
description: Comandos, flags y modos de ejecucion del CLI de pubm.
---

Comando principal:

```bash
pubm
```

Ejemplos:

```bash
pubm patch
pubm minor --preview
pubm 1.8.0
pubm --preflight
pubm --ci
```

## Flag `--snapshot`

- `--snapshot [tag]` — Publica una versión snapshot temporal (etiqueta predeterminada: snapshot)
