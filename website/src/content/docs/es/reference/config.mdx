Campos mas importantes:

- `packages`
- `plugins`
- `versioning`
- `fixed`
- `linked`
- `snapshotTemplate`

Los registros se infieren desde los archivos de manifiesto. `packages[].registries` permite sobreescribirlos o anadir registros privados.

## `snapshotTemplate`

- Tipo: `string`
- Valor predeterminado: `"{tag}-{timestamp}"`

Plantilla para la cadena de versión al usar el flag `--snapshot`.

Variables disponibles:

- `{base}` — versión base actual del archivo de manifiesto
- `{tag}` — etiqueta de snapshot (predeterminada: `snapshot`)
- `{timestamp}` — timestamp Unix en el momento del build
- `{commit}` — hash corto del commit de Git
