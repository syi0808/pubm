---
title: Config API Reference
description: Campos soportados en `pubm.config.*`, valores por defecto y notas de uso.
---

Campos mas importantes:

- `packages`
- `plugins`
- `versioning`
- `fixed`
- `linked`
- `snapshot.useCalculatedVersion`

Los registros se infieren desde los archivos de manifiesto. `packages[].registries` permite sobreescribirlos o anadir registros privados.
