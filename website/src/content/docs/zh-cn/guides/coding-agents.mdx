本指南面向已经在使用编码代理的团队。

## 先做 setup

如果仓库还没有接好，最重要的动作不是 publish，而是 setup。

先使用 `publish-setup`。把 [INSTALLATION.md](/pubm/plugins/pubm-plugin/INSTALLATION.md) 链接提供给代理，让它获取 bundle 并遵循仓库里版本化的 setup 流程。

## Claude Code 安装

```text
/plugin marketplace add your-org/claude-plugins
/plugin install plugin-name@marketplace-name
```

如果不使用 marketplace，就把 `INSTALLATION.md` 链接提供给代理。
