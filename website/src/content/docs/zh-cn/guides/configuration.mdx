`pubm` 是 CLI-first 工具，但真实项目很快就会超出纯 flag 配置。`pubm.config.*` 用来放仓库级 publish 目标、包布局、插件和 changeset 版本规则。

## 支持的配置文件

1. `pubm.config.ts`
2. `pubm.config.mts`
3. `pubm.config.cts`
4. `pubm.config.js`
5. `pubm.config.mjs`
6. `pubm.config.cjs`

## registry 推断规则

| 清单 | 推断的 registry |
|---|---|
| `package.json` | `npm` |
| `jsr.json` | `jsr` |
| `Cargo.toml` | `crates` |

## 常用字段

- `packages`
- `plugins`
- `versioning`
- `fixed`
- `linked`
- `ignore`
- `snapshotTemplate`
