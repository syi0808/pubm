Capabilities

Built for modern package workflows

⚡

Atomic Multi-Registry

npm and jsr publish concurrently. Extend to any registry with a plugin architecture. One command publishes everywhere.

🔐

Smart 2FA Handling

OTP prompts when interactive. Provenance publish when headless. Tokens encrypted locally with multiple security layers.

🛡

Rigid Safety Guards

Branch checks, work-tree validation, remote divergence detection, registry ping, login & permission verification.

👁

Preview & Rollback

Inspect the full task graph with --preview. Automatic rollback on failure. See exactly what will happen before it does.

🔧

Pluggable Pipeline

Customize every step via pubm.config.js. Add custom registries, modify tasks, integrate with your existing workflow.

🤖

CI-Native Design

Prompts auto-disable in CI or non-TTY. NODE_AUTH_TOKEN support. Built for automated pipelines from day one.

Get Started

Up and running in seconds

Install globally
npm i -g pubm
Preview your release
pubm patch --preview
Publish to multiple registries
pubm patch --registry npm,jsr
Or configure once
// pubm.config.js
export default {
  registry: ['npm', 'jsr'],
  branch: 'main'
}

Stop juggling registries.
Start shipping.
